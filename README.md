# vue-better-slider

## 这是什么(What is it)
为vue提供的一个轮播图组件

## 代码演示如何使用

npm install vue-better-slider

在需要的项目文件中导入改组件 import Slider from 'vue-better-slider'

具体代码引用
先注册改组件之后，template模板使用例子如下
```javascript

  <slider>
      <div >
        <a href="">
          <img scr="">
        </a>
      </div>
  </slider> 
```

## 说明

1. 改组件是根据better-scroll滑动组件封装改成
2. 实现功能有：轮播 自动播放 dots 循环播放，适合手机端
